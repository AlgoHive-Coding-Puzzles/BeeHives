--- Quest N part 1 - Hex Colors ---

Il est temps de prendre une décision capitale : de quelle couleur sera le drapeau de la nouvelle Ruche ? Chaque drone abeille a voté pour sa couleur préférée, sans tenir compte des instructions qui demandaient de choisir entre Rouge Bleu ou Vert. La Reine vous demande quoi faire.


Vous proposez de rassembler les couleurs votées selon leur proximité avec le rouge, le vert ou le bleu pour trouver une couleur qui traduise la volonté du peuple. Les proportions de votes pour chacun des trois choix initiaux formeront le code couleur du nouveau drapeau.


Vous devez ensuite rapporter ce code couleur en base 10 pour que Bee-Casso, le peintre royal, puisse présenter sa facture.


Votre input est une liste de couleurs au format hexadécimal, par exemple :

#FFA345
#01FAF3
#45FA01

Calculer la distance euclidienne entre chaque couleur, pour déterminer la couleur la plus proche de chacune des trois couleurs primaires. Par exemple avec "#FFA345" => (255, 163, 69), la distance euclidienne entre cette couleur et le rouge "#FF0000" (255, 0, 0) est :

Rouge = sqrt((255-255)^2 + (163-0)^2 + (69-0)^2) = sqrt(0 + 26569 + 4761) = sqrt(31330) = 177.06
Vert = sqrt((255-0)^2 + (163-255)^2 + (69-0)^2) = sqrt(65025 + 7569 + 4761) = sqrt(77700) = 278.67
Bleu = sqrt((255-0)^2 + (163-0)^2 + (69-255)^2) = sqrt(65025 + 26569 + 3456) = sqrt(93250) = 305.38

La couleur la plus proche de "#FFA345" (255, 163, 69) est le rouge avec une distance de 177.06.


Les couleurs les plus proches de chacune des couleurs de l'input sont :

Hex color: #FFA345, Distances - Rouge: 177.00282483621552, Vert: 279.7320146139873, Bleu: 355.23231834955556 => Rouge
Hex color: #01FAF3, Distances - Rouge: 431.3525240450089, Vert: 243.0534920547327, Bleu: 250.28983199482954 => Vert
Hex color: #45FA01, Distances - Rouge: 311.6039152513973, Vert: 69.18814927427962, Bleu: 363.0110191165001 => Vert

Ce qui donne au niveau des votes :


    {
      "Rouge": 1,
      "Vert": 2,
      "Bleu": 0
    }

En se basant sur le résultat des votes, le code couleur du drapeau sera :


(1, 2, 0), en hexa => (#010200) => en base 10 => 66048

Si vous avez plus de 255 votes pour une couleur, vous devez en faire le modulo 256 pour que Bee-Casso puisse peindre le drapeau.


Vous pouvez passer d'une base à une autre en python grâce à la fonction

int("010200", 16)

qui vous donnera le nombre 66048.


Donnez le code couleur du drapeau en base 10, en vous basant sur les résultats du vote.
