--- Quest 5 Part 1 - Find All Pairs That Sum to Target ---

[ EXPLANATION ]

Dans cette étape, vous devez non seulement trouver les paires `(x, y)` où `x + y = target`, mais aussi vous pouvez également mélanger les nombres des deux colonnes pour former de nouvelles paires, mais n'ajoutez une paire que si elle est unique. Également, vous devez vous assurer que chaque paire est unique. Cela signifie que si une même paire apparaît plusieurs fois (dans un ordre inversé ou dupliquée), elle ne doit être comptée qu'une seule fois (par exemple, `(x, y)` et `(y, x)` sont considérées comme la même paire).



Exemple:

Objectif: 91
(6, 33)
(66, 25)
(72, 19)
(85, 58)
(58, 33)
(33, 58)
Traitement:
// On cherche d'abord les paires entres elles
(6 + 33) = 39
(66 + 25) = 91 => Trouvé
(72 + 19) = 91 => Trouvé
(85 + 58) = 143
(58 + 33) = 91 => Trouvé
(33 + 58) = 91 => Ignoré (déjà trouvé à la symétrique)
// On cherche ensuite les paires sur la colonne de gauche
(6 + 66) = 72
(6 + 72) = 78
(6 + 85) = 91 => Trouvé
(6 + 58) = 64
(33 + 66) = 99
(33 + 72) = 105
(33 + 85) = 118
(33 + 58) = 91 => Ignoré (déjà trouvé à la symétrique)
...
// On cherche ensuite les paires sur la colonne de droite
(33 + 25) = 58
(33 + 19) = 52
(33 + 33) = 66
(25 + 19) = 44
(25 + 33) = 58 => Ignoré (déjà trouvé à la symétrique)
(19 + 33) = 52 => Ignoré (déjà trouvé à la symétrique)
... 4 paires uniques (33, 58), (25, 66), (19, 72), (6, 85)
Sortie attendue : 4

[TEXT]. Affichez la somme totale des paires trouvées.
