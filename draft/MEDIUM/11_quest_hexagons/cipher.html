--- Quest N part 1 - Hexagonal Grid Navigation ---

{ TEXT }

La grille est représentée par une série de mouvements dans les six directions possibles : `N`, `NE`, `SE`, `S`, `SW`, `NW`. Par exemple :

N, NE, SE, S, SW, NW

Chaque cellule hexagonale a une valeur associée. Les ressources ajoutent des points, tandis que les obstacles en retirent. Par exemple : 

    . = 0 points (vide)
    R = +5 points (ressource)
    O = -1 points (obstacle)

Votre tâche est de calculer le score total obtenu par l'abeille après avoir suivi une série de mouvements donnés.

Exemple :

    Grille :
    x R .
    O . R
    . R O
    Mouvements : SE, SE, N, NW

    Calcul :
    - Départ : (0, 0) = . (+0)
    - SE : (1, 1) = . (+0)
    - SE : (2, 2) = O (-1)
    - N : (2, 1) = R (+5)
    - NW : (1, 0) = R (+5)

    Score total : 0 + 0 - 1 + 5 + 5 = 9

Trouvez le score total obtenu par l'abeille après avoir suivi les mouvements donnés dans votre input.
