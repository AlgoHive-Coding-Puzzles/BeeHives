--- Quest N part 1 - Hex Colors ---

Il est temps de prendre une décision capitale : de quelle couleur sera le drapeau de la nouvelle Ruche ? Chaque drone abeille a voté pour sa couleur préférée, sans tenir compte des instructions qui demandaient de choisir entre Rouge Bleu ou Vert. La Reine vous demande quoi faire.


Vous proposez de rassembler les couleurs votées selon leur proximité avec le rouge, le vert ou le bleu pour trouver une couleur qui traduise la volonté du peuple. Les proportions de votes pour chacun des trois choix initiaux formeront le code couleur du nouveau drapeau.


Vous devez ensuite rapporter ce code couleur en base 10 pour que Bee-Casso, le peintre royal, puisse présenter sa facture.
